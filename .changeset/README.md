# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the changesets repo](https://github.com/changesets/changesets).

## How to add a changeset

1. Run `pnpm changeset` in the root of the repository
2. Select the packages you want to include in the changeset
3. Choose a bump type (major, minor, patch)
4. Write a summary of the changes

The changeset will be consumed when running `pnpm version-packages` and the versions will be bumped accordingly.
